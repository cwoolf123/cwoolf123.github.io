Pier One:

This site was made for a local restaurant and was built inside of WordPress to ensure that the site could be updated easily. The concept was to make the entire restaurant menu feature on the site with every element changeable for any future modifications. The title is made using pure CSS3 and SVG graphics. It changes with every title of the page or blog. The auto-scroll feauture allows the user to navigate to a desired menu section quickly. Perfect for placing large images inbetween to add to the User Experience.

Languages used: PHP, HTML5, CSS3, Javascript, jQuery.
